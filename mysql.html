# Mysql 安装

本教程以mysql5.5版本安装为示例

# 下载安装

# 1.首先进入官网 https://www.mysql.com

前往官网下载

MySQL分为社区版和商业版，社区版是免费的，这里选择社区版的。

此时选择一个比较简单的5.5版本

直接下载Wisndow(X86,32-bit)MIS安装版

点击下载，具体安装保存路径可根据自己所需保存位置，这个直接保存到了C盘

# 检验安装

打开命令行界面（cmd），输入mysql -uroot -p,输入自己设置的密码，可以进去mysql界面就是安装成功了！！
